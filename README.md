**Sujet et fonctionnalitées mises en place :**

    Pour ce projet github nous avons choisi de faire un clicker. Le but est d'appuyer sur le bouton central pour générer un maximum de viande. La viande récupérée permet de recruter d'autres loups (améliorations) pour améliorer le nombre de morceaux de viande chassé par seconde. Le but du jeu est d'arriver au plus grand nombre de morceaux de viande possible.

**Principale(s) difficulté(s) rencontré(es):**

    L'une des grosses difficulté que nous avons rencontré était de travailler a 5 sur le même repo. Dès qu'un de nous 5 push sans prévenir, tout les autres ne pouvaient pas push sans pull avant. Cette situation a créer quelques situations ou on avait plusieurs conflits à gérer dans le code. 

    Nous avons tous trouvé que l'utilisation de git était beaucoup plus facile si on était pas plusieurs à travailler dessus en même temps.
